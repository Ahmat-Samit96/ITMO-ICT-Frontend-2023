Install Bootstrap’s source Sass and JavaScript files via npm, RubyGems, Composer, or Meteor. Package managed installs don’t include documentation or our full build scripts. You can also use our npm template repo to quickly generate a Bootstrap project via npm.

Элементы, связанные через aria-describedby, не должны быть видны. Можно сослаться на элемент, даже если этот элемент скрыт. Например, элемент управления формы может иметь скрытое по умолчанию описание, которое отображается по запросу с помощью виджета раскрытия информации, такого как значок «Дополнительная информация». Зрячий пользователь нажимает на значок; для пользователей вспомогательных технологий ссылка на описание осуществляется непосредственно из этого поля формы с помощью aria-describedby.
